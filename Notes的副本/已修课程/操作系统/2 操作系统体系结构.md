## 操作系统服务

1. 用户界面：CLI，GUI，Batch
2. 程序执行：将程序加载到内存，运行，结束执行（包括正常或不正常）
3. I/O 操作
4. 文件系统操作
5. 通信：进程与进程之间，不同计算机之间
6. 错误检测
7. 资源分配
8. 记账
9. 保护与安全

## 系统调用

系统调用提供操作系统服务的接口，通常以 C 或 C++编写

通常，开发人员根据 API 来设计程序

### 向操作系统传递参数

1. 通过寄存器
2. 通过内存里的块、表来传递
3. 堆栈

## 系统调用的类型

1. 进程控制
2. 文件管理
3. 设备管理
4. 信息维护
5. 通信
6. 保护
